如题

前端开发的调试方式

  1. 借助以下插件进行调试。
    • Redux DevTools
    • React Developer Tools
    • React Perf
  2. F12 之后，在源代码处打断点。
  3. 像用 IDEA 调试 Java 代码一样，用 WebStorm 调试 react 代码，直接在 WebStorm 源代码上打断点，调试起来有点象在 IDEA 下调试 Java 程序，很爽！

    • 以 qq 浏览器为例，搜索：Redux DevTools

  2. 添加相应代码以支持在浏览器上使用 Redux DevTools 插件。
