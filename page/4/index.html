    • 如果是在 gitee.com 和 github.com 上同步代码，假设默认仓库放到 github.com，只需要 git remote add 仓库别名 gitee.com上的仓库地址，然后在代码有提交时，执行 git push 仓库别名 即可达到目的，仓库别名 自己起一个好记的就行，比如 gitee。
